# CSS Sidebar Menu

A modern and interactive **sidebar navigation menu built using only HTML and CSS**.  
This project focuses on clean UI, smooth animations, and glassmorphism effects without using any JavaScript.

It demonstrates how far CSS alone can be used to build functional and visually polished UI components.
